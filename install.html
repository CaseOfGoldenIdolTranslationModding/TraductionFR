Installation

Steam

  1. Extraire les fichiers et lancer patch.bat
  2. Répondre 'oui' à la demande d'autoriser l'application à apporter des modifications
  3. Lorsque demandé, choisir le fichier game.pck dans :

    ...\Steam\steamapps\common\The Case of the Golden Idol

    Accessible facilement sur Steam en faisant : Clique droit sur le jeu -> Gérer -> Parcourir les fichiers locaux

GOG

  1. Extraire les fichiers et lancer patch.bat
  2. Répondre 'oui' à la demande d'autoriser l'application à apporter des modifications
  3. Lorsque demandé, choisir le fichier goldenidol.pck dans le dossier où le jeu est installé

Désinstallation

Steam

  1. Clique droit sur le jeu -> Propriétés -> Fichiers installés -> Vérifier l'intégrité des fichiers du jeu

GOG

  1. Retélécharger le jeu sur GOG

Attention

  • Chaque mise à jour du jeu enlève la version française, je ferais en sorte de mettre à jour la version française mais il faudra quand même la réinstaller à chaque fois.
  • Pour installer une version plus récente de la version française, il faut d'abord désinstaller la version française.
  • Votre progression sans ou avec la version française n'est pas la même car elle est sauvegardée sur des fichiers différents pour éviter des problèmes d'incompatibilité. Donc si vous avez déjà joué au jeu et que vous comptez le faire en français, vous devrez recommencer de zéro, mais votre progression sur le jeu normal n'est pas supprimée.
